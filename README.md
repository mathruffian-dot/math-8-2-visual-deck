# 國中數學八下整冊視覺化互動簡報

這是一份依據「114 國中數學 2 下」教材章節整理的原創互動 HTML 簡報。

## 使用方式

- 直接開啟 `index.html`
- 拖拉左側單元卡到右側焦點區
- 使用滑桿改變視覺模型
- 用左右方向鍵快速切換單元

## 涵蓋單元

- 第 1 章：等差數列、等差級數、等比數列、費氏數列
- 第 2 章：函數與函數圖形、生活中的訊號
- 第 3 章：三角形內外角、尺規作圖、三角形全等、中垂線與角平分線、三角形邊角關係、角度面面觀
- 第 4 章：平行、平行四邊形、特殊四邊形、發現平行之旅

## 素材說明

公開版只使用章節架構與原創視覺化圖形，不上傳教科書 PDF 或頁面截圖。
