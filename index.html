重點 1：三角不等式 (Triangle Inequality)

任意兩邊之和大於第三邊

是不是任意給定三根竹籤，就一定可以排出三角形呢？答案是否定的。 在二維平面上，三條線段要首尾相接拼成一個封閉圖形，必須滿足「任意兩邊之長大於第三邊」的約束。

📐 數學核心定理：
設三角形的三邊長分別為 $a$、$b$、$c$，則必滿足：
❶ $a + b > c$      ❷ $b + c > a$      ❸ $a + c > b$
同理可推得：任意兩邊之差的絕對值必小於第三邊。

重點 2：等腰三角形性質 (Isosceles Triangle)

等邊對等角 ＆ 等角對等邊

等腰三角形是幾何中極具對稱美的圖形。當一個三角形中有兩條邊相等時，這兩條腰所夾的對角（底角）也必定相等。 反過來說，如果三角形中有兩個內角相等，則它們的對邊也必定相等。

📐 數學核心定理：
若線段 $\overline{AB} = \overline{AC}$，則其對角 $\angle B = \angle C$。
此性質稱為「等邊對等角」。反向論證「等角對等邊」亦成立。

重點 3：大角對大邊，大邊對大角

三角形內邊角不等關係

在同一個三角形中，如果三邊長不相等，那麼邊與角之間有著絕對的對應關係： 較長的那條邊，它所面對的那個角也必定是最大的；較大的那個角，它所對應的邊也必定是最長的。

📐 數學核心定理：
在 $\triangle ABC$ 中，若 $a > b > c$，則其對角必滿足 $\angle A > \angle B > \angle C$。
此定理僅適用於「同一個三角形內」或「等大三角形之間」的邊角比較。

重點 4：幾何探索 — 大邊對大角證明

摺紙法推導邊角不等關係

如何嚴謹地說明「較長的邊對應較大的角」呢？我們可以使用教科書最經典的**「摺紙對稱與外角性質」**進行直觀的幾何推導。

📐 幾何推導步驟：
❶ 摺疊對稱：已知 $\overline{AB} > \overline{AC}$。我們將右腰 $\overline{AC}$ 沿著角平分線 $\overline{AD}$ 向左摺疊，使 $C$ 點落在 $\overline{AB}$ 上的 $E$ 點。摺疊後 $\triangle ACD \cong \triangle AED$，故 $\angle 1 = \angle C$ 且 $\overline{AE} = \overline{AC}$。
❷ 外角性質：觀察 $\triangle BED$。因為 $\angle 1$ 為其頂點 $E$ 的外角，故必滿足 $\angle 1 > \angle B$。
❸ 不等代換：因為 $\angle C = \angle 1 > \angle B$，故得證 $\angle C > \angle B$！

重點 5：幾何探索 — 大角對大邊證明

垂直平分線對摺與三角不等式

若三角形中「角大於角」，那它們所對應的邊，是否也一定是「邊大於邊」呢？我們可以使用**「底邊垂直平分線對摺」**來進行嚴謹證明。

📐 幾何推導步驟：
❶ 垂直平分對摺：已知 $\angle C > \angle B$。我們沿著底邊 $\overline{BC}$ 的垂直平分線 $\overline{DE}$ 摺疊，使 $B$ 點與 $C$ 點重合，摺痕為 $\overline{DE}$。由對稱性可知 $\triangle CDE \cong \triangle BDE$，故 $\overline{DC} = \overline{DB}$ （式①）。
❷ 三角不等式：觀察 $\triangle ACD$。根據三角形「兩邊之和大於第三邊」的性質，必滿足：$\overline{AD} + \overline{DC} > \overline{AC}$ （式②）。
❸ 代入求證：將式①代入式②，得 $\overline{AD} + \overline{DB} > \overline{AC}$。由於 $D$ 點落在 $\overline{AB}$ 上，即 $\overline{AD} + \overline{DB} = \overline{AB}$，故得證 $\overline{AB} > \overline{AC}$！

重點 6：外角性質 (Exterior Angle Property)

外角等於內對角之和

將三角形的任意一邊向外延伸，就可以得到外角。 三角形的任意一個外角，會剛好等於與它不相鄰的兩個內角（內對角）相加之和，且必定大於任何一個單一內對角。

📐 數學核心定理：
在 $\triangle ABC$ 中，延長 $\overline{BC}$ 邊至 $D$，形成外角 $\angle ACD$，則：
❶ $\angle ACD = \angle A + \angle B$
❷ $\angle ACD > \angle A$   且   $\angle ACD > \angle B$
頂角 A 大小 50°
底角 B 大小 60°

重點 7：挑戰小測驗 (Interactive Quiz)

檢驗您的學習表現

恭喜您完成了所有重點概念的探索！請花幾分鐘完成下列互動挑戰，測試自己是否完全掌握了「三角形的邊角關係」。

Q1. 設三角形的三邊長為 $4$、$9$、$x$，且 $x$ 為整數，則 $x$ 有幾種可能？
解析：根據三角不等式，兩邊之差 $< x <$ 兩邊之和。
故 $9 - 4 < x < 9 + 4 \implies 5 < x < 13$。
因為 $x$ 為整數，符合的有 $6, 7, 8, 9, 10, 11, 12$ 共 7 種，故選 B。
Q2. 在 $\triangle ABC$ 中，已知 $\angle A = 40^\circ$，$\angle B = 80^\circ$，則下列哪條邊最長？
解析：首先求出第三個角 $\angle C = 180^\circ - 40^\circ - 80^\circ = 60^\circ$。
故三個角的大小順序為 $\angle B (80^\circ) > \angle C (60^\circ) > \angle A (40^\circ)$。
根據「大角對大邊」，最大的角 $\angle B$ 所對的邊 $\overline{AC}$ 最長，故選 C。
Q3. 三角形一外角為 $115^\circ$，且此角的一內對角為 $45^\circ$，則另一內對角的大小為何？
解析：根據「外角等於兩內對角之和」，可設另一內對角為 $x^\circ$。
則 $45^\circ + x^\circ = 115^\circ \implies x^\circ = 70^\circ$。故選 B。
